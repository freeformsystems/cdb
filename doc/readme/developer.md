## Developer

### Test

The [rlx][] module tests this library extensively.

### Lint

Run the source tree through [jshint][] and [jscs][]:

```
npm run lint
```

### Import

Imports configuration data from `default.ini` into `config.js`:

```
npm run import
```

### Readme

To build the readme file from the partial definitions:

```
npm run readme
```
